# DGentic 0.1.0 Release Notes

Release date: 2026-05-07

## Summary

DGentic 0.1.0 is the first backend foundation release. It provides a FastAPI service skeleton, shared Pydantic contracts, a deterministic starter task planner, tests, and documentation for the initial implementation path.

## Included

- FastAPI application factory and server entrypoint.
- Installed CLI command: `dgentic-server`.
- Health endpoints:
  - `GET /`
  - `GET /health`
- Task planning endpoint:
  - `POST /tasks/plan`
- Core schemas for:
  - Task requests
  - Task plans
  - Plan steps
  - Step results
  - Providers
  - Agent briefs
  - Tool manifests
  - Log events
- Deterministic starter planner.
- Backend tests.
- Repository architecture documentation.
- Developer setup documentation.
- Reserved `localmcp/` directory for future generated tools.

## Install From Wheel

```powershell
python -m venv .release-test
.\.release-test\Scripts\python -m pip install dist\dgentic-0.1.0-py3-none-any.whl
```

## Run

```powershell
dgentic-server --host 127.0.0.1 --port 8000
```

Open:

- `http://127.0.0.1:8000/health`
- `http://127.0.0.1:8000/docs`

## Known Limitations

- No model provider integrations yet.
- No filesystem jail or CLI policy engine yet.
- No task execution loop yet.
- No memory index or vector retrieval yet.
- No web UI or VS Code extension yet.
- The planner is deterministic and does not call AI models.
