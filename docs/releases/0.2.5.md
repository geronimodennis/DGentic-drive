# DGentic 0.2.5 Release Notes

Release date: 2026-05-07

## Summary

DGentic 0.2.5 releases the README implementation status policy update. The root README now explicitly separates implemented, partially implemented, and not-yet-implemented features so every sprint and release has a clear project status checkpoint.

## Included

- Updated the root README `Current Status` section with:
  - Implemented features.
  - Partially implemented features.
  - Not-yet-implemented features.
- Added the README status policy requiring the status section to be updated after every sprint, release, or meaningful implementation change.
- Bumped package, API, backend `__version__`, and generated tool default version metadata to `0.2.5`.
- Updated documentation index and release distribution guide.
